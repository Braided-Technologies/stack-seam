# Project Memory

## Core
StackSeam — IT Stack Intelligence Portal for MSPs. Brand: Slate+Amber (#F59E0B). Domain: stackseam.tech.
Fonts: Syne (headings), Outfit (body), DM Mono (data). Lovable Cloud backend.
Multi-tenant: org-based data isolation via RLS. Roles: admin, member, platform_admin.
React Flow for interactive stack integration map.
